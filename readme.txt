== Description ==

inax is a "buy SimCard Credit For Irancell, MCI and Rightel Operators" support package for Wordpress.

Features include:

*	Buy Topup SimCard Credit
*	Buy Pin SimCard Credit
*	Buy Intetnet Package For Iranian operators

== Installation ==

You can install inax by either using wordpress.org plugin directory, or by directly uploading plugin files to your server.
